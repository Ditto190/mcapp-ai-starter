### Version 0.16.0 (Release Date: February 11 2026)

- **Status**: Public Preview
- **Enhancements**:
  - Updated hosted sample to include "Seattle Hotels" single agent
  - Add project endpoint directly to .env file
  - Introduced Agent Inspector to improve local run experience for hosted agents. (Press F5 to start agent inspector)
  - Added Agent Builder for prompt agents, users can edit, deploy, and test prompt agents directly in VS Code.
  - Cost warnings removed and added more contextual messaging when deploying models and hosted agents.
  - Released log streaming to surface hosted agent logs in VS Code.
  - Simplified starter template project structure to streamline local run experience.
  - Agent details/playground surfaced automatically after deployment.
